# HTML5 Video Conversion Helpers

To use:

    bundle install
    brew bundle
    ./convert path/to/movie.mov  output-name-without-extension

This will output the following files:

    output/
      index.html
      player.swf
      output-name.jpg
      output-name.webm
      output-name.mp4

player.swf was lovingly stolen from <https://github.com/csscallum/HTML5-Video>
